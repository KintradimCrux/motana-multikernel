### Arguments:

**method:**

* Name: method
* Is required: no
* Is array: no
* Description: Test method
* Default: `NULL`

### Options:

**all:**

* Name: `--all`
* Shortcut: `-a`
* Accepts value: no
* Is value required: no
* Is multiple: no
* Description: Run all tests
* Default: `false`

**file:**

* Name: `--file`
* Shortcut: `-f`
* Accepts value: yes
* Is value required: no
* Is multiple: no
* Description: File to process
* Default: `NULL`

**verbose:**

* Name: `--verbose`
* Shortcut: `-v|-vv|-vvv`
* Accepts value: no
* Is value required: no
* Is multiple: no
* Description: Increase the verbosity of messages: 1 for normal output, 2 for more verbose output and 3 for debug
* Default: `false`
